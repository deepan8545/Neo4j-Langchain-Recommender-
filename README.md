# Neo4j + LangChain Explainable Recommendation System

This project builds an explainable course recommendation system using:
- Neo4j (Graph Database)
- LangChain + OpenAI (LLM explanations)

## Graph Model

(:User)-[:PURCHASED {rating, discount_purchase}]->(:Product)

## Dataset

- 100 Users
- 6 Products
- 500 purchase interactions
- Rating (1–5)
- Discount purchase flag (0/1)

## Setup

1. Install dependencies:
   pip install -r requirements.txt

2. Configure .env file based on .env.example

3. Load data:
   python src/load_to_neo4j.py

4. Run recommendation:
   python src/recommend.py --user User_12

## Output

System returns:
- Recommended products
- Natural language explanation generated by LLM
